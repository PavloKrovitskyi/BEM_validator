# FLS БЕМ валідатор v0.2.3

Клієнтський HTML-валідатор на відповідність методології **БЕМ**. Аналізує код, знаходить помилки в іменуванні та вкладеності класів, будує візуальне дерево розмітки.

## Як працює

1. Вставте HTML у текстове поле → натисніть **«Перевірити»**.
2. Валідатор парсить HTML через DOMParser, рекурсивно обходить вузли та аналізує класи.
3. Знайдені помилки групуються за класом — кожна з описом порушення та шляхом у DOM.
4. Якщо помилок немає — повідомлення про успіх і персонаж Жека знизу.
5. За наявності помилок будується дерево розмітки з підсвіченими проблемними класами.

## Правила валідації (10 типів помилок)

| Код | Суть |
|-----|------|
| `noParentBlock` | Елемент `block__elem` поза своїм блоком |
| `recursiveElement` | Елемент вкладено в однойменний елемент |
| `recursiveBlock` | Блок вкладено в однойменний блок |
| `onlyModifier` | Модифікатор без блока/елемента |
| `elementOfElement` | Подвійний `__` в імені |
| `moreThanOneBlock` | Два блоки на одному вузлі |
| `moreThanOneElement` | Два елементи на одному вузлі |
| `hierarchy` | Спроба ієрархії через дефіс у назві |
| `onlyClosestParent` | Елемент не від найближчого блоку |
| `parseError` | Невалідний HTML |

## Дерево розмітки

- Відображає структуру HTML з тегами та класами.
- Класи з помилками підсвічені золотим — **клік** по них прокручує до помилки та анімує її спалахом.
- Клік по будь-якому класу підсвічує його кольором.
- Повзунок **«Рівні вкладеності»** ховає гілки нижче заданого рівня.
- Автоматично визначає код цілої сторінки (додає `<html>`/`<body>` у дерево).

## Інтерфейс

- Темна тема (CSS-змінні, закоментована світла).
- Сітка з двох колонок: результати + дерево.
- Кнопка **«Вгору»** після прокрутки 100+ px.
- Чистий ES6, жодних залежностей, повністю на стороні клієнта.
